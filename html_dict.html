Dicionário de Tags HTML

Tag Descrição
<!DOCTYPE html> Define o tipo de documento e a versão do HTML.
<html> Elemento raiz do documento HTML.
<head> Contém metadados, links para estilos e scripts.
<meta> Define metadados do documento, como charset e viewport.
<title> Define o título da página exibido na aba do navegador.
<link> Usado para conectar folhas de estilo e fontes externas.
<body> Contém o conteúdo visível da página.
<header> Define um cabeçalho para a página ou seção.
<nav> Define uma área de navegação com links.
<ul> Cria uma lista não ordenada.
<li> Define um item dentro de uma lista.
<a> Cria um link para outra página ou recurso.
<div> Elemento genérico para estruturação e estilização.
<aside> Define conteúdo relacionado, como uma barra lateral.
<img> Insere imagens na página.
<h1> - <h2> Definem títulos e subtítulos.
<p> Define parágrafos de texto.
<section> Agrupa conteúdo relacionado.
<main> Define o conteúdo principal da página.
<footer> Define um rodapé para a página.
<address> Fornece informações de contato.
<table> Cria uma tabela.
<th> Define um cabeçalho de coluna dentro da tabela.
<td> Define uma célula dentro de uma tabela.
<tr> Define uma linha dentro da tabela.
